mintUpload by emorrp1

Uploads files on the Internet

mintUpload allows you to upload files on the Internet. This makes it easier to share or send big files which would not fit in an email.

Dependencies

python (>= 2.5), python (<< 3), python-gtk2, python-glade2, mintsystem, python-pexpect, python-paramiko

Install

The latest dev snapshot is available as a deb. Following the Linux Mint policy, only RC-quality versions are released, so it should be stable, please report any problems you find: add this to /etc/apt/sources.list : 

deb http://ppa.launchpad.net/emorrp1/ppa/ubuntu jaunty main
then force version in synaptic
